# PSO-University-Timetable-Scheduling

## GUI instructions 

## Usage 
1. Create virtual enviromemnt with python version 3.10
example with conda:
```
conda create -p venv python=3.10
```
2. Activate venv
```
conda activate venv
```

## Installation
```
pip install -r requirements.txt
```

## Run App
```
streamlit run gui.py
```
